# Node Web App - Linux DevOps Practice

A simple Node.js Express application deployed on Ubuntu Linux.
This project demonstrates real-world DevOps skills:
- Linux server setup
- Node.js deployment
- Nginx reverse proxy
- Systemd services
- Bash automation
- Cron jobs
- Logging and monitoring
- CI/CD integration

---

## Features
- Basic Express app with one route
- Runs on port 3000
- Can be deployed on Linux VM or cloud
- Ready for automation and CI/CD pipelines
